# Fuji.me!

Lightweight Fuji lens and camera explorer with genre-based scoring.

**Domain:** [fujime.app](https://fujime.app)
**Part of the me! series by [braboj.me](https://braboj.me)**

## Stack

- **Framework:** Astro (static output) + React islands
- **Language:** TypeScript (strict mode)
- **Styling:** Astro scoped styles + CSS Modules
- **Deployment:** GitHub Pages via GitHub Actions

## Status

Work in progress — data migration phase. See [docs/dev-summary.md](docs/dev-summary.md) for details.

## Development

```bash
npm install
npm run dev       # localhost:4321
npm run build     # production build
npm run preview   # preview build
npm test          # run tests
```
